**Weather**

|         |                                                |
|---------|------------------------------------------------|
| **d20** | **Temperature**                                |
| 1-14    | Normal for the season                          |
| 15-17   | 1d4 x 10 degrees Fahrenheit colder than normal |
| 18-20   | 1d4 x 10 degrees Fahrenheit hotter than normal |
| **d20** | **Wind**                                       |
| 1-12    | None                                           |
| 13-17   | Light                                          |
| 18-20   | Strong                                         |
| **d20** | **Precipitation**                              |
| 1-12    | None                                           |

| 18-20   | Heavy rain or heavy snowfall                   |
